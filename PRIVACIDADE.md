# Política de Privacidade - MeetPerto
**Última atualização: 3 de julho de 2026**

 O MeetPerto leva a sua privacidade a sério.

## 1. Dados que coletamos
- **Cadastro** Nome, e-mail, data de nascimento 18+, gênero, cidade
- **Perfil**  Até 7 fotos que você envia voluntariamente
- **Localização** Usada apenas para mostrar pessoas próximas. Sua posição exata nunca é exibida para outros usuários
- **Uso** Dados** de interação para segurança anti-fake

## 2. Como usamos seus dados
Exclusivamente para operar o app: mostrar perfis compatíveis, garantir segurança e previnir fraudes. Não vendemos seus dados.

## 3. Compartilhamento
Não compartilhamos dados com terceiros para marketing. Usamos Firebase/Google apenas para infraestrutura, sob contrato de proteção de dados.

## 4. Seus direitos LGPD
Você pode solicitar a qualquer momento: acesso, correção, exclusão de dados ou revogar consentimento pelo e-mail: contatomeetperto.app

## 5.  Segurança
Senhas com hasg Argon2/bcrypt. Conexão HTTPS em todo app. Tokens  com expiração.

## 6. Exclusão da conta
Em configurações > Excluir conta. Todos os dados são apagados permanentemente em até 30 dias.
